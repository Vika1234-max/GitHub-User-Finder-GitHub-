# GitHub User Finder

**Автор:** Мелихова Виктория

## Описание

Приложение для поиска пользователей GitHub с возможностью сохранения избранных профилей. Использует GitHub API для получения данных и сохраняет избранное в JSON‑файл.

## Функционал

- Поиск пользователей GitHub по имени.
- Отображение результатов в виде таблицы (логин, имя, ссылка на профиль).
- Добавление пользователей в избранное.
- Сохранение избранного в JSON‑файл между запусками.
- Проверка на пустой ввод.

## Как использовать API GitHub

Приложение использует публичный GitHub API:

- **Endpoint:** `https://api.github.com/users/{username}`
- **Метод:** GET
- **Ограничения:** Без аутентификации — 60 запросов в час. Для увеличения лимита используйте токен.
