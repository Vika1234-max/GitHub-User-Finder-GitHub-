# Movie Library - Личная кинотека

## Автор
**Мелихова Виктория**

---

## Описание

**Movie Library** — это графическое приложение для ведения личной кинотеки. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга, фильтровать коллекцию по жанру и году, а также редактировать и удалять записи. Все данные сохраняются в JSON-файл и загружаются при запуске.

### Основные возможности:

- ➕ **Добавление фильмов** с проверкой ввода
- 📋 **Отображение** всех фильмов в таблице
- 🔍 **Фильтрация** по жанру и году выпуска
- ✏️ **Редактирование** существующих записей
- 🗑️ **Удаление** фильмов из коллекции
- 💾 **Автоматическое сохранение** в JSON
- ✅ **Валидация данных** (год — число, рейтинг 0-10)
- 📊 **Статистика** (количество фильмов, средний рейтинг, популярные жанры)
- 🎨 **Интуитивно понятный интерфейс**

## Установка и запуск

### Требования
- Python 3.7 или выше
- tkinter (входит в стандартную поставку Python)

### Инструкция

1. **Скачайте или клонируйте проект**
```bash
git clone https://github.com/ekaterina-sokolova/movie-library.git
cd movie-library
Запустите приложение

bash
python main.py
Как использовать
Добавление фильма
Заполните поля:

Название — любое название фильма

Жанр — выберите из выпадающего списка

Год выпуска — число от 1888 до текущего года

Рейтинг — число от 0 до 10 (можно с десятичной точкой)

Нажмите кнопку «Добавить фильм»

Фильтрация
По жанру — выберите жанр из выпадающего списка

По году — введите год в поле «Год»

Нажмите «Применить фильтр»

Для сброса фильтра нажмите «Сбросить фильтр»

Редактирование
Выберите фильм в таблице

Нажмите «Редактировать выбранный»

Измените данные в открывшемся окне

Нажмите «Сохранить»

Удаление
Выберите фильм в таблице

Нажмите «Удалить выбранный фильм»

Подтвердите удаление

Примеры использования
Пример 1: Добавление фильма
Входные данные:

Название: Побег из Шоушенка

Жанр: Драма

Год: 1994

Рейтинг: 9.3

Результат: Фильм появляется в таблице с рейтингом 9.3 ⭐⭐⭐⭐☆

Пример 2: Фильтрация по жанру
Действия:

Выбрать жанр Комедия

Нажать «Применить фильтр»

Результат: Отображаются только комедии

Пример 3: Фильтрация по году
Действия:

Ввести год 1999

Нажать «Применить фильтр»

Результат: Отображаются фильмы только 1999 года

Пример 4: Ошибка валидации
Входные данные:

Рейтинг: 11

Результат: Ошибка «Рейтинг должен быть от 0 до 10!»

Пример 5: Редактирование фильма
Действия:

Выбрать фильм «Побег из Шоушенка»

Нажать «Редактировать»

Изменить рейтинг на 9.5

Сохранить

Результат: Рейтинг обновляется в таблице

Рекомендуемые тесты
Тест 1: Корректное добавление
Вход: Название=«Матрица», Жанр=«Фантастика», Год=1999, Рейтинг=8.7

Ожидаемый результат: Фильм добавлен, появляется в таблице

Тест 2: Неверный год (буквы)
Вход: Год=«абв»

Ожидаемый результат: Ошибка «Год должен быть числом!»

Тест 3: Неверный год (диапазон)
Вход: Год=1500

Ожидаемый результат: Ошибка «Год должен быть от 1888 до 2025»

Тест 4: Неверный рейтинг
Вход: Рейтинг=15

Ожидаемый результат: Ошибка «Рейтинг должен быть от 0 до 10!»

Тест 5: Пустое название
Вход: Название=«» (пусто)

Ожидаемый результат: Ошибка «Введите название фильма!»

Тест 6: Фильтрация без результатов
Вход: Жанр=«Вестерн», Год=2025

Ожидаемый результат: Сообщение «Фильмы не найдены!»

Тест 7: Добавление нескольких фильмов
Действие: Добавить 5 фильмов

Ожидаемый результат: Счётчик статистики показывает 5 фильмов

Тест 8: Редактирование
Действие: Изменить название фильма

Ожидаемый результат: Название обновляется в таблице

Тест 9: Удаление фильма
Действие: Удалить фильм

Ожидаемый результат: Фильм исчезает из таблицы, статистика обновляется

Тест 10: Сохранение в JSON
Действие: Добавить фильм, закрыть и открыть приложение

Ожидаемый результат: Фильм остаётся в коллекции
